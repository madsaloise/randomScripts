Quick Start

  1. Interactive Map: Open portfolio.html to explore the project dependency graph (toggle between Map and List views)
  2. Browse Projects: Click any category above or any project in the map to view detailed documentation
  3. Navigate Between Projects: Click dependencies/dependants to seamlessly jump between related projects
  4. File Share Compatible: All pages work without a web server - perfect for network shares

Features

  • ✅ Dual view modes: Interactive Map & Organized List
  • ✅ Dynamic project detail pages (single source of truth)
  • ✅ Clickable lineage visualization with dependency graphs
  • ✅ Color-coded categories for easy navigation
  • ✅ Cross-linked documentation with seamless navigation
  • ✅ Works on file:// URLs (no server needed)
  • ✅ Responsive design for all screen sizes
